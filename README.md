
## Description

Crud opertion, User authentication and authorization functionalities.
PS: Test Case of getting all cats and getting a cat by Id

## Install Packages

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test:dev
```
